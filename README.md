# Employee Management System

## Overview
This project is a Java application that demonstrates the core concepts of Object-Oriented Programming (OOP) through an Employee Management System.

The system includes two types of employees:
- Manager
- Programmer

Both employee types share common information such as employee ID, name, and salary, but each calculates bonuses using its own method.

## Objectives
- Implement encapsulation using private fields and getters/setters.
- Use inheritance to create employee subclasses.
- Demonstrate constructor overloading.
- Override methods to calculate bonuses differently for each employee type.
- Apply runtime polymorphism through parent class references.

## Features
- Store employee details
- Calculate bonuses for Managers and Programmers
- Display employee information and calculated bonuses
- Demonstrate Java OOP principles

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## Project Structure
```
src/
├── Employee.java
├── Manager.java
├── Programmer.java
└── Main.java
```

## How to Run
1. Clone this repository.
2. Open the project in your preferred Java IDE.
3. Compile and run `Main.java`.
4. View the employee details and calculated bonuses in the console.

## OOP Concepts Demonstrated
- Encapsulation
- Inheritance
- Constructor Overloading
- Method Overriding
- Runtime Polymorphism
